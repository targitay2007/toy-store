# Toy Store Website

A modern, responsive e-commerce website showcasing a cool toy. Built with HTML, CSS, and JavaScript.

## Features

- Hero section with product image and call-to-action
- Featured products grid with hover effects
- Categories section
- About page
- Newsletter signup
- Shopping cart demo
- Fully responsive design

## Deployment

To deploy this site to GitHub Pages:

1. Create a new repository on GitHub (e.g., `toy-store`).
2. Clone the repository locally:
   ```bash
   git clone https://github.com/yourusername/toy-store.git
   cd toy-store
   ```
3. Copy the contents of this folder (`index.html`, `style.css`, `script.js`) into the repository.
4. Commit and push:
   ```bash
   git add .
   git commit -m "Initial commit: toy store website"
   git push origin main
   ```
5. Go to repository Settings → Pages.
6. Under "Source", select the `main` branch and `/ (root)` folder.
7. Click Save. Your site will be published at `https://yourusername.github.io/toy-store/`.

## Customization

- Edit `index.html` to change content, images, or product details.
- Modify `style.css` for colors, fonts, and layout.
- Adjust `script.js` for additional interactivity.
